Requirements: 

**Java JDK 1.8+  
MySQL 5.6+**  

To run:  

1. config your DB prop: [jdbc.properties](https://github.com/SergeyVolovyk/UniMenu/blob/master/service/src/main/resources/jdbc.properties)
1. create DB `cafe`  
1. run `./gradle update` to setup your DB  
1. run `./gradle run` to stats server
