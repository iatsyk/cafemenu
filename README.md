# cafemenu

Requirements: 

**Java 8+  
Gradle 2.7+  
MySQL 5.6+**  

To run:  

1. config your DB prop: [jdbc.properties](https://github.com/iatsyk/cafemenu/blob/master/src/main/resources/jdbc.properties  )
1. create DB `cafe`  
1. run `gradle update` to setup your DB  
1. run `gradle run` to stats server
